# 🛒 Tienda de Don Tino Comics

E-commerce for comics and manga built with React and Firebase.

🔗 [Live demo](https://tdt-comics.vercel.app/)

> Built in 2023 as part of my React learning path.You said: ponlo con cdigo htmll asi se lee bien

![Screenshot](./screenshot.png)

## 🛠️ Stack

- React
- JavaScript
- Firebase Firestore
- Material UI
- SCSS
- React Router
- Vercel

## ✨ Features

- Browse by publisher (Marvel, DC Comics, Ivrea)
- Filter by title (Batman, Spiderman, X-Men, etc.)
- Shopping cart with localStorage persistence
- Checkout with order generation in Firestore
- Product pagination
- Skeleton loading states

## 🚀 Local setup

```bash
git clone https://github.com/AFCoiro/TDT_Comics
cd TDT_Comics
npm install
npm start
```
